Protein Molecular Dynamics Setup (BioBB based)
by Universitat de Barcelona

Setup protocol for preparing a protein system for running a GROMACS moleculara dynamics. The pipeline is using a set of opensource BioExcel Building Blocks.

Steps:

  • Fix Protein Struture.
  • Create Protein System Topology.
  • Create Solvent Box
  • Fill the box with Water Molecules.
  • Adding Ions.
  • Energetically Minimize the System
